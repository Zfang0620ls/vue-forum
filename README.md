# vue-forum
#### 利用vuejs中文社区提供的api重写的webapp
#### 感谢https://github.com/shinygang的vue-cnodejs项目给予我的启发，app的UI设计也来自于此，十分感谢

## 安装
```
$ git clone https://github.com/zyl1314/vue-forum.git
$ cd vue-forum
$ npm install
```
## 运行
```
$ npm run dev
```
